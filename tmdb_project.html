Introduction¶

This data set contains information of about 10,000 movies, collected from The Movie Databse(TMDb). It contains information regarding the cast, director, production company, release date, revenue, ratings, genre, playtime and more.

Questions for research

  • Which are the most popular, and least popular genres based on the number of movies in each genre in our database
  • How have user ratings for movies in popoular genres, changed over 10 year periods. Keeping in mind the the average rating of movies falling in the studied genres in the 10 year periods.
  • How have the user ratings for popular genres changed over these periods, keeping in mind the average user sentiment for each genre over 50 years.
  • How have the profits for movies in popular genres, changed over 10 year periods in our database.

Dependent Variable: YEARBINS

a : 1960-1969 b : 1970-1979 c : 1980-1989 d : 1990-1999 e : 2000-2009 f : 2010-2019

Independent Variables: Ratings of popular genres over yearbins.

def isnan(f):
    return isinstance(f, float) and f!=f

def iszero(q):
    if isnan(q):
        return False
    if q ==0:
        return True
    else:
        return False

Conclusions¶

Limitations:

  • The database is just a sample. We dont have all movies from 1960-2015. Sample may not be the best estimate of the population.
  • Rows for which values were missing, were removed from all calculations.

After grouping by year_bin, and finding average rating of genres studied in each yearbin; it is interesting to note that the average rating of movies in our database have been on a constant decline from the 1960s until now. This maybe because the sample of movies in our database is not the best estimate of the population of movies from 1960s until now.
